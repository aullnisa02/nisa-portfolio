Proyek Pengembangan Website

Peran Saya di Pengembangan Website

Dalam proyek-proyek pengembangan website statis ini, saya menerapkan pengetahuan dasar saya dalam HTML, CSS, dan JavaScript untuk membangun antarmuka pengguna yang bersih, responsif, dan fungsional. Fokus utama saya adalah pada struktur yang semantik, styling yang estetis sesuai tema, dan interaktivitas dasar untuk meningkatkan pengalaman pengguna. Saya memastikan setiap proyek dioptimalkan untuk berbagai ukuran layar dan memiliki performa yang baik.

Peran Utama: Web Developer, Web Designer, Frontend Developer

Tools Utama: VS Code, HTML5, CSS3, JavaScript, GitHub

Tipe Proyek: Portofolio Pribadi, Undangan Digital

Tahun: 2025

Berikut adalah beberapa contoh website statis yang pernah saya bangun:

Invitation Wedding Digital for Indry & Fahrul

Saya merancang dan mengembangkan website undangan pernikahan untuk Indry & Fahrul dengan desain yang elegan dan romantis, menggunakan HTML, CSS, dan Bootstrap agar responsif dan mudah diakses di berbagai perangkat. Website ini menampilkan fitur-fitur seperti halaman hero dengan countdown menuju hari H, informasi detail acara, galeri foto pasangan, timeline cerita cinta, bagian RSVP interaktif, serta fitur doa dan ucapan dengan animasi halus. Seluruh elemen dirancang untuk memberikan pengalaman pengguna yang nyaman dan menyenangkan, sekaligus menghadirkan nuansa personal sesuai tema pernikahan. Proyek ini menunjukkan kemampuan saya dalam menggabungkan desain visual, interaktivitas, dan pengembangan front-end untuk kebutuhan acara spesial.

Web Dummy for Coding Camp: Website Perusahaan

Saya merancang dan mengembangkan website perusahaan ini dengan desain yang simple dan formal, menggunakan HTML, CSS, dan JavaScript agar responsif dan mudah diakses di berbagai perangkat. Seluruh struktur halaman dibangun secara manual tanpa framework, agar saya dapat lebih memahami dasar-dasar pengembangan web. Website ini terdiri dari beberapa bagian penting seperti landing page, profil perusahaan, lokasi layanan, portfolio, dan form kontak. Untuk tampilan visual, saya menggunakan kombinasi warna biru dan abu-abu untuk menciptakan kesan profesional dan terpercaya. Selain itu, saya juga menerapkan prinsip mobile-first design agar website tetap nyaman digunakan di perangkat smartphone maupun tablet. Melalui dummy project ini, saya belajar banyak tentang layouting dengan CSS Flexbox dan Grid, penggunaan DOM JavaScript untuk interaktivitas dasar, serta pentingnya struktur semantik dalam HTML agar ramah SEO dan mudah dibaca oleh browser maupun pembaca layar. Project ini menjadi latihan penting dalam mengasah keterampilan front-end saya dan menunjukkan kemampuan saya dalam membangun website sederhana namun fungsional dari awal hingga selesai.
